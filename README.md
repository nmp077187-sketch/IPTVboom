# Плейлист IPTV – IPTVboom

Этот репозиторий содержит разные вариации IPTV-плейлиста в формате .m3u, которые можно использовать в VLC, IPTV Smarters и других медиаплеерах. Рекомендую использовать TVirl + Live Channels


## Доступные версии

- Полный плейлист

```
https://freaktv.ru/IPTVboom.m3u
```

- Урезанный
  
```
https://freaktv.ru/IPTVboom-.m3u
```

- Только детские каналы

```
https://freaktv.ru/IPTVboomKids.m3u
```

- Только белорусские

```
https://freaktv.ru/IPTVboomBel.m3u
```
